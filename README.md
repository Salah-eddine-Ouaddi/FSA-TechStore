# FSA TechStore 🛍️

Une boutique en ligne dédiée aux étudiants et professeurs de la **Faculté des Sciences d'Agadir** (FSA).

## 📋 Description du Projet

**FSA TechStore** est une application web e-commerce permettant aux utilisateurs de :
- 📝 S'inscrire et se connecter à leur compte
- 🔍 Parcourir un catalogue de matériel informatique
- 🛒 Sélectionner des produits
- 📄 Générer des factures professionnelles
- 📥 Télécharger les factures en PDF

Cette boutique propose une sélection de matériel informatique adapté aux besoins académiques avec des prix accessibles.

## ✨ Fonctionnalités Principales

- **Authentification utilisateur** : Inscription et connexion sécurisées
- **Catalogue de produits** : 9 articles de matériel informatique
- **Panier d'achat** : Sélection multiple de produits via cases à cocher
- **Génération de factures** : Factures personnalisées avec informations client
- **Export PDF** : Téléchargement des factures en format PDF
- **Interface responsive** : Adaptée aux différentes tailles d'écran
- **Navigation intuitive** : Menu de navigation simple et clair

## 🛠️ Technologies Utilisées

- **HTML5** : Structure et mise en page
- **CSS3** : Stylisation et design responsive
- **JavaScript (Vanilla)** : Logique côté client et gestion des états
- **Aucune dépendance externe** : Application légère et autonome

## 🚀 Installation et Utilisation

### Prérequis
- Un navigateur web moderne (Chrome, Firefox, Safari, Edge, etc.)

### Étapes d'installation

1. **Cloner le dépôt**
   ```bash
   git clone https://github.com/votre-nom/FSA-TechStore.git
   cd FSA-TechStore
   ```

2. **Ouvrir l'application**
   - Ouvrir le fichier `FSA TechStore.html` dans un navigateur web
   - Ou servir les fichiers via un serveur web local

3. **Utiliser l'application**
   - Cliquer sur "S'inscrire" pour créer un compte
   - Se connecter avec vos identifiants
   - Parcourir le catalogue de produits
   - Sélectionner les articles souhaités
   - Générer et télécharger votre facture en PDF

## 💾 Stockage des Données

Actuellement, l'application stocke les données des utilisateurs en mémoire locale (variables JavaScript). Les données sont réinitialisées lors du rechargement de la page.

**Note** : Pour une utilisation en production, il est recommandé d'intégrer une base de données backend.
